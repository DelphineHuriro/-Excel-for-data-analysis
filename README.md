#  Excel for data analysis
This project is aimed to help people interested in data science jobs across the globe by showing salaries for different job titles,required skills,jobs platforms where jobs are posted.
## Salary dashboard
This dashboard was designed to help job seekers filter jobs by specifying which job title they are interested in,country where they would like to work in and what job schedue they are interested in.By specifying those,they can see mediam salary,top job platform and jobs count.
Check it out here [Dashboard](https://github.com/DelphineHuriro/-Excel-for-data-analysis/tree/main/Dashboard%20project)
